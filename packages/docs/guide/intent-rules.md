# Intent Rules (Removed)

> **This feature has been removed.** Intent Rules were replaced by the simpler [Oversight System](/guide/oversight) in v0.5.0.
>
> The Oversight System provides three tiers (Supervised / Balanced / Autonomous) plus natural language custom instructions, replacing the per-category allow/alert/review rule matrix.
>
> See the [migration guide](/guide/oversight#migration-from-intent-rules) for how to translate your old rules.
